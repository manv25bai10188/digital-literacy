## Project Overview

This project documents my journey as a Student Digital Ambassador at VIT Bhopal University. The objective was to build a comprehensive digital literacy portfolio to help peers navigate professional online spaces, secure communication, and cyber safety. Over five modules, I have developed resources ranging from visual infographics to technical safety checklists, all organized within this central GitHub repository to demonstrate "proof of work" in the digital age.

## Module Summaries

### [Task 1: Digital Literacy Awareness Infographic](./task-1-presentation/Digital%20Literacy.png)

Created a modular infographic using **Canva** covering three critical pillars: Useful Digital Tools, Safe Internet Practices, and Email Etiquette. It provides quick, actionable advice for students, such as utilizing Password Managers and professional email salutations.

### [Task 2: Student Digital Portfolio](./task-2-portfolio/)

Established a professional "digital twin" by setting up and optimizing profiles on **GitHub**, **LinkedIn**, and **Kaggle**. These platforms host code, manage professional networks, and track AI/ML contributions.

### [Task 3: Coding & Collaboration Platforms](./task-3-platforms/)

Practiced technical skills on **HackerRank** (Python challenges) and streamlined data collection using **Google Workspace**. Designed a "Digital Literacy Awareness Quiz" using Google Forms to engage peers.

### [Task 4: Email Etiquette & Communication](./task-4-email-etiquette/)

Developed formal communication standards with sample emails for extension requests and internship inquiries. Included a **Social Media Checklist** to ensure a professional and safe digital footprint.

### [Task 5: Cybercrime Awareness](./task-5-cybercrime/)

Researched Ransomware trends in the Indian education sector and developed an **automated "assume-breach" backup strategy**. Includes a case study and a prevention checklist for cyber safety.
